Out
===

### ActionScript 3 Log Engine ###

Created by [Joel Ray](https://github.com/joelray).


### Basic Usage

Out.registerEngine(Trace.getInstance());
Out.filterLevel = Out.DEBUG;
Out.filterByEquality = false;
Out.clear();
Out.info(this, "A simple log setup!");
